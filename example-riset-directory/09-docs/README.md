# 09-docs

Folder ini berisi dokumen rencana penelitian saya yang disusun sesuai topik penelitian aktual: analisis pengaruh algoritma weighted probability terhadap peluang rare item pada sistem gacha game.

## Isi Utama

- [rencana-penelitian.md](rencana-penelitian.md) — ringkasan penelitian, rumusan masalah, tujuan, variabel, dan roadmap kerja
- [tahap-1-arsitektur-dan-skema-database.md](tahap-1-arsitektur-dan-skema-database.md) — desain eksperimen, struktur data simulasi, dan parameter penelitian
- [tahap-2-implementasi-gateway.md](tahap-2-implementasi-gateway.md) — implementasi simulator gacha dan logika probabilitas
- [tahap-3-pengujian-k6.md](tahap-3-pengujian-k6.md) — eksekusi simulasi berulang dan pengumpulan data mentah
- [tahap-4-analisis-data.md](tahap-4-analisis-data.md) — analisis statistik, cumulative probability, dan visualisasi hasil
- [tahap-5-draf-paper.md](tahap-5-draf-paper.md) — struktur penulisan naskah jurnal dan status penyusunan

## Dokumen Pendukung

- [archive/promt.md](archive/promt.md) — arsip prompt awal yang dipakai saat menyusun arah penelitian
- [../.ai/research-context.md](../.ai/research-context.md) — konteks singkat penelitian untuk pendampingan AI

## Alur Kerja yang Dipakai

1. Buka [rencana-penelitian.md](rencana-penelitian.md) untuk melihat peta penelitian keseluruhan.
2. Ikuti tahapan secara berurutan dari desain eksperimen sampai penulisan naskah.
3. Simpan hasil kerja pada folder terkait: [../05-kode](../05-kode), [../04-data](../04-data), [../06-output](../06-output), dan [../07-manuskrip](../07-manuskrip).
