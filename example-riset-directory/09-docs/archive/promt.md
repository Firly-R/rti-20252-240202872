# Arsip Prompt Awal Asisten Riset

Dokumen ini menyimpan prompt awal yang dipakai untuk mengarahkan asisten riset dalam menyusun roadmap penelitian yang sesuai dengan topik saya.

## Ringkasan Prompt

- Fokus penelitian: analisis pengaruh algoritma weighted probability terhadap peluang rare item pada sistem gacha game.
- Objek penelitian: simulasi sistem gacha dengan fixed probability dan weighted probability.
- Tujuan eksperimen: membandingkan cumulative probability serta rata-rata pull untuk memperoleh rare item.
- Deliverable utama: desain eksperimen, implementasi simulator, data simulasi, analisis, dan draf paper.

## Catatan

Isi dokumen ini dipertahankan sebagai arsip sejarah pengerjaan, sementara detail aktif disimpan pada dokumen tahap yang lebih spesifik di folder 09-docs.
